# Diffractive-deep-optical-neural-network-D2NN

This repository includes two *diffractive-deep-optical-neural-networks* (D2NNs) trained in TensorFlow and PyTorch, along with the scripts for generating femtosecond-laser fabrication files.

- **Training**: Deep learning with NPCC loss and Angular Spectrum Propagation
- **Fabrication**: Femtosecond-laser two-photon polymerization (TPP)

#### Reference

D2NNs were first introduced by Ozcan et al. in the Science 2018 paper (https://www.science.org/doi/10.1126/science.aat8084), which established the framework of using trainable diffractive layers to implement deep optical neural networks.
